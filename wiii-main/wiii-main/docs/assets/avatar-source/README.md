# Avatar Source Assets

This folder stores source PNGs and reference exports for the Wiii avatar.

Use it for:
- concept or iteration exports
- source images used to design avatar expressions
- reference material that should be preserved in git but is not loaded directly by the app

Runtime assets should stay in the relevant app folders such as `wiii-desktop/public/animations/` or `wiii-desktop/src-tauri/icons/`.

Retained source references:
- `wiii-avatar-1.png`
- `wiii-avatar-2.png`
- `wiii-avatar-3.png`
- `wiii-avatar-4.png`

These files are intentionally kept as avatar concept/source references. The
broken 16-byte `wiii-avatar.png` placeholder was removed because it was not a
valid image or a referenced runtime asset.
